Instituto Superior Tecnológico del Azuay Información de la aplicación Bienvenidos a nuestra aplicación "Student Notes", el objetivo del presente proyecto es crear una aplicación para dispositivos móviles que se encargue de mostrar las notas de cada estudiante pertenecientes a cualquier carrera del Instituto Superior Tecnológico del Azuay (ISTA), las cuales estarán disponibles para docentes y alumnos.
Para llevar a cabo este objetivo se utilizará los datos ingresados por los docentes en la base de datos del sistema "Phoenix" creado especialmente para los docentes de la institución. Objetivo General El objetivo general de este proyecto es el desarrollo de una aplicación para dispositivos móviles el cual permitirá de forma más cómoda y rápida el acceso a la información de las notas de los estudiantes. Facilitando asi, cualquier inconveniente o consulta que desee realizar el estudiante. Objetivos Específico - Desarrollar una aplicación móvil que permita consultar las notas meidante un previo registro del usuario.
- Mostrar información de las notas del ciclo mediante la información guardada en la base de datos del sistema.
